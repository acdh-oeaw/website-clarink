**The  WBÖ Document Database**&#x20;

The database includes about 85% of all entries of the Dictionary of Historical Bavarian Dialects in Austria and South Tyrol (WBÖ) starting with the letter D. It is based on the WBÖ paper slip catalog. It comprises around 2.4 million entries and has been publicly accessible via the Lexical Information System Austria (LIÖ) since 2020. The paper slips have been digitized by typing from 1993 until 2011, initially in TUSTEP format; since 2014, the data has been converted into XML/TEI format in several iterative steps. This dataset is available via: [https://lioe.dioe.at/db](https://lioe.dioe.at/db).

**The LexAT21 Dataset**

This dataset consists of all answers to questionaires of the LexAT21 questionaire rounds. This data was collected all over Austria via written online questionaires and asks the informants to write down their variants for different concepts in both their standard and nonstandard varieties. The dataset is available via LexAT21 and as regular dumps after every survey round via the repository ARCHE. This dataset is available via: [https://lexat21.lapis-online.at/en/db?a=0,100\&q=11\&r=all\&v=all](https://lexat21.lapis-online.at/en/db?a=0,100\&q=11\&r=all\&v=all).

**The DIAUMA Dataset**

The Digitising Austrian Dialect Maps dataset is available via the DIAUMA online interface or the ARCHE repository. It consists of roughly 800 TIFF files of language and dialect maps as well as metadata on the individual maps. This dataset is available via [https://diauma.acdh.oeaw.ac.at/index/](https://diauma.acdh.oeaw.ac.at/index/).
