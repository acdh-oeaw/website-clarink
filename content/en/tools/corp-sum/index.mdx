CorpSum is a web application that enables user-friendly, dynamically generated queries in text corpora along different extralinguistic extralinguistic dimensions of variation (such as the dimensions time and space). Find out more here: [https://www.oeaw.ac.at/acdh/research/dh-research-infrastructure/resources/applications/corpsum-visualization-tool-for-corpus-requests](https://www.oeaw.ac.at/acdh/research/dh-research-infrastructure/resources/applications/corpsum-visualization-tool-for-corpus-requests).
